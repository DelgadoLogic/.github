## How We Work

| Principle | What it means |
|-----------|---------------|
| **Source-first** | We compile everything ourselves. No pre-built third-party binaries in critical paths. |
| **Privacy by default** | Features are opt-in. Telemetry requires explicit consent. |
| **Built to last** | We support legacy Windows where others have walked away. |
| **Sovereign updates** | All update manifests are Ed25519-signed. You can verify before applying. |
| **AI-native** | AI features run locally first. Cloud is always opt-in. |
